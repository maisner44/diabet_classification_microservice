Використання API

Тільки GET запити

  • http://127.0.0.1:8000/api/patients - для отримання списку всіх пацієнтів додатку
  • http://127.0.0.1:8000/api/doctor/doctorID/patients - для отримання списку всіх пацієнтів конкретного доктора

Тільки POST

  • http://127.0.0.1:8000/api/create-patient - для створення нового пацієнта

  • POST запит повинен приймати такі payloads:

    "username", "password", "date_of_birth", "first_name", "last_name", "patronymic", "height", "weight", "diabet_type", "email", "phone_number"

Ендпоінти для записів у картку пацієнта ( GET та POST методи )

  • http://127.0.0.1:8000/api/patient/username/glucose-measurements - створення запису по глюкозі
  • http://127.0.0.1:8000/api/patient/username/physics - створення запису по фізичній активності
  • http://127.0.0.1:8000/api/patient/username/insuline-dose - створення запису по ін'єкції інсуліну
  • http://127.0.0.1:8000/api/patient/username/foods - створення запису по прийому їжі
